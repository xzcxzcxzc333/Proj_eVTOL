# Proj_eVTOL

面向多电飞机（More Electric Aircraft, MEA）与 eVTOL 电推进系统的科研项目仓库。

本项目用于沉淀电推进、电机驱动、能量管理与系统建模相关的研究资料、代码和分析结果。

## 当前内容

- `codex_reading_notes/`：eVTOL 电驱动综述文献的提取文本、索引、元数据与阅读导图。

## 约定

- 后续项目代码、仿真模型与可复现实验脚本将持续提交至本仓库。
- 原始资料与生成结果请按主题分目录存放，并在新增模块时补充使用说明。

## 计划方向

- 电推进系统建模与参数分析
- 电机及其控制策略研究
- 能量管理与多电系统协同优化
- 仿真验证与数据分析
